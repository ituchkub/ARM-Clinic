เรียน ผู้เกี่ยวข้อง

_CompanyTH_ (“บริษัท”) ขอแจ้งให้ทราบว่าขณะนี้ได้พ้นกำหนดระยะเวลางดซื้อขายหลักทรัพย์ (Blackout Period) แล้ว

อย่างไรก็ตาม บริษัทขอย้ำว่าแม้ไม่อยู่ในช่วงเวลางดซื้อขายหลักทรัพย์ (Blackout Period) ก็ตาม ท่านยังต้องปฏิบัติตามกฎเกณฑ์ที่เกี่ยวข้องกับการใช้ข้อมูลภายใน รวมถึงแต่ไม่จำกัดเพียงพระราชบัญญัติหลักทรัพย์และตลาดหลักทรัพย์ และ แนวปฏิบัติเรื่องการใช้ข้อมูลภายในของบริษัทต่อไป ทั้งนี้ หากท่านดำรงตำแหน่งกรรมการและ/หรือผู้บริหารของบริษัท และมีความประสงค์จะซื้อขายหลักทรัพย์ของบริษัทฯ และ/หรือ บริษัทในกลุ่มบริษัทฯซึ่งเป็นบริษัทจดทะเบียน ท่านจะต้องแจ้งให้ส่วนเลขานุการคณะกรรมการบริษัททราบอย่างน้อย _LeastDay_ วันล่วงหน้าก่อนทำการซื้อขาย ตามนโยบายการกำกับดูแลกิจการที่ดีของบริษัท

1. กรอกแบบฟอร์ม และส่งกลับมาที่ Email: _CompanyEMail_ คลิก _Download_ หรือ
2. แจ้งทางช่องทางอิเล็กทรอนิกส์ อื่น เช่น แอพพลิเคชั่น LINE เป็นต้น
